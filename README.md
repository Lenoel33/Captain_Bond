# Captain Bond 007 Copy

React + Vite + Tailwind recreation of the Captain Bond public site flow using the components you pasted.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Build

```bash
npm run build
```

## Notes

- The original Base44 backend/database is not included because it is not public.
- Registration submissions are saved to browser localStorage through `src/api/base44Client.js`.
- Main editable files are in `src/components` and translations are in `src/lib/LanguageContext.jsx`.
- Image URLs still point to the public Base44 media links from your copied components.
